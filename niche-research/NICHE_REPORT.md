# Отчёт: анализ ниш и подниш YouTube по живым данным API

**Дата:** 12 июля 2026 · **Метод:** YouTube Data API v3, только реальные цифры
**Профиль:** англоязычный фейсли/AI-аватар канал, цель — быстрая монетизация

## Методика

- 53 подниши в 7 нишах (психология, научпоп, наука, технологии, деньги, бизнес, финансы)
- 53 поисковых запроса → **1 718 уникальных каналов**, из них **788 малых** (<50k подписчиков)
- Скачаны последние загрузки всех 788 каналов → **28 924 видео** с полными метаданными
- Учитывались **только видео, вышедшие после 12.01.2026** (последние 6 месяцев) и **только длинные** (>3 мин)
- **Критерий успеха канала:** <50k подписчиков + 10 последних длинных роликов за 6 мес + ≥7 из 10 набрали 30 000+ просмотров
- Все цифры перепроверены повторными запросами к API (расхождение — единицы просмотров, живой прирост)

**Главный результат:** из 625 активных малых каналов жёсткий фильтр прошли **6** (5 англоязычных). Это значит: почти везде малые каналы не набирают просмотров, и ценность — именно в тех поднишах, где новички стабильно пробиваются.

## Итоговая рекомендация

**Основной вход — №1 POV-сторителлинг (скамы/хакеры/аферы)**: лучшая проходимость (2 победителя из 9 активных), медианы 68–176k у каналов младше года, идеально для фейсли + AI, минимальная себестоимость (сценарий + озвучка + AI-картинки).

**Страховочный/парный вариант — №3 Деньги × сторителлинг**: выше RPM, два канала-доказательства младше 9 месяцев, темы бесконечные, и это пересечение твоих интересов (психология + деньги). Можно тестировать оба параллельно: производственный конвейер один и тот же.

## Ограничения и следующий шаг

- Суточная квота поиска API исчерпана (53 запроса). Второй раунд (усиление подниш-лидеров ещё ~20 запросами) доступен после сброса квоты (~10:00 МСК завтра) — уточнит проходимость топ-3.
- Транскрипты видео в этой среде недоступны (закрытый API Google + сетевая блокировка youtube.com); классификация — по полным заголовкам/описаниям/тегам/главам 28 924 видео.
- Все сырые JSON-ответы API сохранены и приложены к данным исследования.
