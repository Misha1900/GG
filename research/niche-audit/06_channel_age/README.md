# Поправка — анализ по реальному возрасту канала, не по дате ролика

Повод: пользователь указал, что предыдущий разбор "новички набирают больше просмотров" был некорректным — метрика M4 (свежесть) мерила дату публикации ролика, а не дату создания канала. Старый канал (10+ лет) может выпустить один свежий ролик и попасть в выборку как "свежий", хотя это не имеет отношения к тому, может ли реальный новичок пробиться.

## Метод

1. Извлёк уникальные ID каналов (channel_id, не имя) из всех сырых выгрузок по 8 проверенным нишам — `niche_rows.json` (286 уникальных каналов).
2. Запросил `channels.list(part=snippet)` пачками по 50 ID — `channel_created.json` (дата создания каждого канала, поле `publishedAt`). 6 юнитов квоты.
3. Отфильтровал видео по каждой нише: оставил только те, где канал создан после 2025-07-11 (младше 12 месяцев на дату аудита 2026-07-11).
4. Пересчитал долю "молодых" каналов и вытащил конкретные примеры.

## Файлы
- `channel_ids.txt` — список ID для запроса
- `channel_created.json` — дата создания каждого канала
- `niche_rows.json` — все строки по нишам с channel_id (промежуточные данные)
